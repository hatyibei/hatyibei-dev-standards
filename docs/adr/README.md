## ADRテンプレート

```markdown
# ADR-NNN: タイトル

## ステータス
proposed | accepted | deprecated | superseded

## コンテキスト
なぜこの決定が必要か。

## 決定
何を決定したか。

## 根拠
なぜこの選択肢を選んだか。参考にしたリポジトリや手法。

## 影響
この決定によるトレードオフ。
```
